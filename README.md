# AI Voice Assistant

An AI-powered desktop voice assistant built with Python that can understand voice commands, engage in natural conversations, perform system automation tasks, and provide spoken responses through an interactive graphical user interface.

## Features

* Wake Word Detection
* Speech Recognition
* AI-Powered Conversations using Groq LLM
* Text-to-Speech Responses
* Open Applications
* Open Websites
* System Automation
* Interactive GUI
* Real-Time Voice Interaction

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Groq API
* Tkinter / CustomTkinter
* Threading

## Supported Commands

* Open Chrome
* Open WhatsApp
* Open GitHub
* Open LinkedIn
* Open VS Code
* Open Camera
* Open Paint
* Search the Web
* Shutdown PC
* Restart PC

## Capabilities

* Answer user questions
* Hold natural conversations
* Execute voice commands
* Open desktop applications
* Launch websites
* Provide spoken responses
* Perform automation tasks

## Project Structure

```text
main.py         - Main application entry point
brain.py        - Command processing logic
voice.py        - Speech recognition module
wakeword.py     - Wake word detection
apex_gui.py     - GUI interface
ai_engine.py    - AI response generation
groq_test.py    - Groq API testing
```

## How to Run

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python main.py
```
