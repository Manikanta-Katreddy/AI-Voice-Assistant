# AI Voice Assistant

An AI-powered voice assistant built with Python that can understand voice commands, respond intelligently, perform system automation tasks, and interact with users through a graphical interface.

## Features

* Wake Word Detection
* Speech Recognition
* AI-Powered Conversations
* Text-to-Speech Responses
* Open Applications
* Open Websites
* System Automation
* GUI Interface
* Real-Time Voice Interaction

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Groq API
* Tkinter / CustomTkinter
* Threading

## Capabilities

* Answer user questions
* Hold natural conversations
* Execute voice commands
* Open desktop applications
* Launch websites
* Provide spoken responses
* Perform basic automation tasks

## Project Structure

* main.py – Main application
* brain.py – Command processing
* voice.py – Speech recognition
* wakeword.py – Wake word detection
* apex_gui.py – GUI interface
* ai_engine.py – AI response generation
* groq_test.py – AI model testing

## Future Enhancements
